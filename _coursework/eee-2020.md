# Course Information
- **Semester:** Fall 2018
- **Credits:** 3
- **Target Audience:** EE Undergraduates
- **Prerequisites:** Understanding of basic programming

# Objectives and Overview
Understand abstract data types, object-oriented programming, and analysis of algorithms.

After successfully taking the class you will acquire:
1. Knowledge to simplify your ideas in abstract data types.
2. Designing algorithms and basic analysis of algorithm performance.

# Topics

1. Review of programming                                                                      
2. Algorithm Analysis                                                                         
3. Lists                                        
4. Stack and Queue  
5. Tree 
6. Hashing / Priority Queue
7. Sorting                                        
8. Graph
